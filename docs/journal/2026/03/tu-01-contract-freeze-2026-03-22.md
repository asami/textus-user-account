# TU-01 Contract Freeze (2026-03-22)

Status: DONE
Owner: textus-user-account

## Scope

User account lifecycle contract for `textus-user-account` generated by Cozy/CNCF.

## Decisions

1. Command/Query boundary
- Command: `create*`, `save*`, `update*`, `delete*`, `delete*Hard`
- Query: `load*`, `search*`

2. Error/failure expectation
- Missing required input is treated as `RecordNotFound`/missing-property style consequence failure in generated request builders.
- Unknown/invalid identifiers remain runtime validation failures in CNCF operation execution path.

3. Package contract (frozen)
- Generated Component package is configurable from model metadata (`COMPONENT` section `package` field).
- EntityValue family package layout is fixed to `${basePackage}.entity/*`.
  - Plain: `${basePackage}.entity.<Entity>`
  - Create/Read/Update/Query/Operation: `${basePackage}.entity.<kind>.<Entity>`
  - Aggregate/View: `${basePackage}.entity.aggregate...` / `${basePackage}.entity.view...`

4. Compatibility policy
- No parallel generation API is added.
- Existing model path is extended in-place (`kaleidox` COMPONENT metadata -> `cozy` Modeler -> `simple-modeler` generation).
